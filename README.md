# TargetSixStrike

## 项目描述
这是一个使用Three.js开发的3D射击游戏，玩家需要在限定时间内射击6个球体目标。游戏采用第一人称视角，包含辅助瞄准、移动控制和HUD显示。目标会随机生成在墙上，避免重叠，并限制在天空高度以下。

## 安装
1. 确保你的浏览器支持WebGL。
2. 无需额外安装依赖，直接打开`index.html`即可运行。

## 如何运行
1. 使用Python启动本地服务器：
   ```bash
   python3 -m http.server 8000
   ```
2. 在浏览器访问 `http://localhost:8000/`。
3. 点击“开始”按钮进入游戏。

## 控件
- **鼠标移动**：控制视角旋转。
- **左键点击**：射击。
- **A/D键**：左右移动。
- **V键（按住）**：启用辅助瞄准。
- **Esc键**：退出指针锁定。

## 游戏规则
- 每轮游戏持续指定时间（默认60秒）。
- 射击目标以得分，目标大小根据难度调整。
- HUD显示剩余时间、命中、未命中和准确率。
- 游戏结束后显示总结，并可重新开始。

## 开发笔记
- 使用Three.js进行3D渲染。
- 实现了deltaTime-based动画以确保平滑性。
